Clickme

ClickMe will help you to create a test page for Click jacking vulnerability

Download as .zip Download as .tar.gz View on GitHub

About

If a website is vulnerable to Clickjacking ,then target web page can be loaded into an frame. To do this we need to create a simple web page that includes a frame containing the target web page. This can be done using Click Me in easy steps.

Usage

1.Give the target site address.

2.Hit the Click Me button.

3.That's it! Now please check whether your site is getting loaded in a frame.

Please refer OWASP web site to know more about Clickjacking vulnerability.
